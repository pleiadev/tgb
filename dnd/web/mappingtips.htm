  1. Choose a background. There is a blank parchment that is a free download from Wizards.
  2. Next make the dungeon layout. Dungeon Crafter 2 is an easy tile tool to use for interiors or villages.
  3. Take a print screen (Alt+Print Screen) of the dungeon and paste it onto the Photoshop project.
  4. Use the Magic Wand (W) to remove everything from the screen shot except the tiles.
  5. Outline the map using the Stroke command to create the outer walls.
  6. Add the Drop Shadow effect.
  7. Add some images behind the tiles, Fade them to 15-20%, and Fade the borders into the parchment.

Drop Shadows

  1. To do this create a duplicate of the layer with the image or text to drop shadow.
  2. Move the image a few pixels to the side and a few pixels up or down from the original.
  3. Adjust the Hue/Saturation (Ctrl+U), check Colorize, and set the Lightness value to -100.
  4. Gaussian Blur (Filter) the layer with a value of about 3.
  5. Adjust the layer's Opacity (Layer Options) to perhaps 60-70% until the desired effect is achieved.

Fading

  1. Select the image layer to fade.
  2. Select white as the background color, and make the image taller with Canvas Size (Image | Canvas Size).
  3. Increase the height by about an inch while leaving the width the same and click the box for the Top-Middle anchor.
  4. Leave about an inch of blank space at the bottom of the image that will be trimmed off later.
  5. Use the lasso tool to draw around the part you want to fade at the top, but be sure to bring it all the way down to the bottom white part.
  6. Feather the selection by about 40 pixels and Invert the Selection (Ctrl+Shift+I).
  7. Clear (Backspace) the selection, and Deselect (Ctrl+D). There should still be the white space at the bottom of the image.
  8. Using the Crop tool, drag it from the top left corner to the bottom right corner of the image, not the white space. After cropping the image, the white space should be gone.

Outlining

  1. Create a new Layer with the image or text you want to outline.
  2. Turn Preserve Transparency off and Stroke (Edit|Stroke) the layer with a value of 1-2. Be sure either outside or center is selected in your location setting.
