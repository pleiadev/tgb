d20-xml Monster Manual Formatter

d20-xml Monster Manual Formatter

This utility can be used to format d20-xml Monsters in compressed statistic block format. Designed to be useful for DMs working on campaigns or encounters for 3rd Edition Dungeons and Dragons game from Wizards of the Coast. This application should be viewed from Internet Explorer. The Monster Manual data file is about 1.7 MBs (very large) so this page runs best when it is run on the local machine. To use the page:

  • Save it to some location on your computer.
  • Save the data file for d20-xml Monster Manual v. 1.3 to the same location. If the Angelfire download keeps timing out try downloading the zip file or get the d20-xml group's zip file.
  • Open the saved page.
  • Select desired monster.
  • Print or Copy Paste as needed.
  • Go Play!

Room Furnishings Generator

Room Furnishings Generator

This utility generates random room descriptions that can be used in dungeon, castle, or town setting and is inspired from the furnishings and tables in the first edition of the Advance Dungeons & Dragons Dungeon Masters Guide written by Gary Gygax. This utility is distributed at no cost and comes with no guarantee or warranty.

Designed to be useful for DMs working on adventures or encounters for 3rd Edition Dungeons and Dragons game from Wizards of the Coast. Feel free to use this in your campaign and pass it along to your friends. Hopefully, it will add a little flair to all those other rooms in your dungeon that you just didn't have time to spend detailing.

Generated rooms do not include traps, monsters, valuable magic items, etc… It is focused on adding mundane items that do not have a large gold piece value yet speed the development of more detailed and varied environments. For the DM's that like to load up every room in the dungeon you can always load up your favorite random trap, monster, or magic item generator and just include it with the description.

The utility avoids being overly specific in regards to layout of items and uses relative terms like "Over there" (where?) or "Against the wall" (which wall?). For example, it doesn't include descriptions of doors and windows so the DM is free to customize the entry for a particular room either in their favorite word processor or on the fly.

The older Excel version of this utility can be found at Room Furnishings Generator.

Random Name Generator

Random Name Generator

Designed to be useful for DMs working on adventures or encounters for 3rd Edition Dungeons and Dragons game from Wizards of the Coast. Feel free to use this in your campaign and pass it along to your friends. Hopefully, it will add a little flair to your dungeon that you just didn't have time to spend detailing.

This utility generates random names for use in your campaign. It is inspired by Chris Pound's name generator at http://www.ruf.rice.edu/~pound who did the real work of coming up with the algorithms. I highly suggest you visit his site to obtain a dataset to use with this file. To adapt a new name set use the Name Generator Compiler located in this zip file.

An older Excel version of this utility can be found at Random Name Generator.
